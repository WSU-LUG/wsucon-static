  • Game: Team Fortress 2
  • Game Version: Latest version available via Steam
  • Game Types: Capture the Flag (CTF) / Push (Linear Control Point) / Stopwatch (Attack/Defend)
  • Time per round: 20 min
  • Number of Maps: 1
  • Map Rounds: 2
  • Number of Teams: 2
  • Players per Team: 5 (min. 4)

  • Respawn Time: Instant respawn (i.e. mp_respawnwavetime 0)
  • Criticals Enabled: No
  • Sudden Death: No

Maps: Turbine, Mach4 Format: Each team plays two 20 minute halves.Teams will switch sides at the end of the each half.
How to Win: Highest score determines a winner. Score is determined by summing the points of both rounds.
Overtime: In case of a tie, an overtime will be played. The first team to score a point (capture the flag) wins.
